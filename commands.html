Termucmd

Здесь все команды

Основные команды

1. pkg install python - Установка пакета (Вместо python можете что нибудь другое)

2. pkg update - Обновление списка доступных пакетов

3. pkg upgrade - Обновление всех установленных пакетов

4. pkg search python - Поиск пакетов (Вместо python можете что нибудь другое)

5. pkg remove python - Удаление пакетов (Вместо python можете что нибудь другое)

6. apt - Это также как и pkg

Работа с файлами и директориями

7. ls - Показывает содержимое текущей директории

8. cd folder - Перейти в указанную директорию (Вместо folder можете что нибудь другое)

9. pwd - Показывает текущею директорию

10. mkdir folder - Создает директорию (Вместо folder можете что нибудь другое)

11. rm file.txt - Удаляет файл (Вместо file.txt можете что нибудь другое)

12. rm -r folder - Удаляет директорию (Вместо folder можете что нибудь другое)

13. cp file.txt /folder/ - Копирует файл или директорию (Вместо file.txt /folder/ можете что нибудь другое)

14. mv file.txt /folder/ (Вместо file.txt /folder/ можете что нибудь другое)

15. touch file.txt - Создает файл (Вместо file.txt можете что нибудь другое)

Работа с текстом

16. cat file.txt - Показывает содержимое файла (Вместо file.txt можете что нибудь другое)

17. nano file.txt - Открывает файл в текстовом редакторе (Вместо file.txt можете что нибудь другое)

18. echo hello - Пишет текст (Вместо hello можете что нибудь другое)

19. grep hello file.txt - Ищет текст в файле (Вместо hello file.txt можете что нибудь другое)

Сеть

20. ping google.com - Проверяет доступность хоста (Вместо google.com можете что нибудь другое)

21. curl google.com - Скачивает или отправляет данные по сети (Вместо google.com можете что нибудь другое)

22. wget google.com - Скачивает файл из интернета (Вместо google.com можете что нибудь другое)

23. ssh user@192.168.1.1 - Подключается к удаленному серверу по SSH (Вместо user@192.168.1.1 можете что нибудь другое)

Процессы

24. ps - Показывает запущенные процессы

25. top - Аналог диспечера задач

26. kill 1234 - Завершает процесс по его ID (Вместо 1234 можете что нибудь другое)

Система

27. uname -a - Показывает информацию о системе

28. df -h - Показывает информацию о свободном месте на диске

29. exit - Закрыть termux или завершить текущую сессию
